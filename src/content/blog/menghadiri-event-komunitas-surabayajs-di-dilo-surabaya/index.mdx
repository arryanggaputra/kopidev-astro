Hai guys, pada [event komunitas SurabayaJS](https://kopi.dev/2021-01-menghadiri-event-komunitas-surabayajs-di-dilo-surabaya/) kali ini saya datang untuk pertama kalinya. Dan pada tulisan ini saya ingin sedikit bercerita (bukan review) tentang acara meetup di [komunitas SurabayaJS](https://kopi.dev/2021-01-menghadiri-event-komunitas-surabayajs-di-dilo-surabaya/) pada tanggal 18 Januari 2020.

## Opening

Seperti yang sudah-sudah, acara event atau meetup komunitas apapun pasti diselingi oleh pembukaan, rasanya ini sudah menjadi ritual khusus disetiap acara komunitas. Dan pembukaan pada meetup di komunitas SurabayaJS ini dibuka oleh Grico sebagai salah satu pengurus di komunitas SurabayaJS, dan setelah itu acara intinya langsung dimulai.

![grico membuka acara event komunitas surabayajs](./images/grico%20membuka%20acara.png "grico membuka acara event komunitas surabayajs")

Event komunitas SurabayaJS yang saya datangi kali ini bertema **Machine Learning using TensorFlow.js and jQuery Alternatives,** yang mana materinya dibawakan oleh langsung oleh Dzulfikar Adi Putra (_jQuery Alternatives_), dan Rakha Asyrofi (_Machine Learning using TensorFlow.js_).

Saya perhatikan peserta yang datang di acara [event komunitas SurabayaJS](https://kopi.dev/2021-01-menghadiri-event-komunitas-surabayajs-di-dilo-surabaya/) ini cukup banyak, menurut penerawangan saya pesertanya lebih dari 50, angka 50 adalah jumlah minimal yang dapat saya terawang, untuk jumlah pastinya, silahkan bertanya pada pengurum komunitas SurabayaJS.

![peserta Event bulanan komunitas SurabayaJS ](./images/peserta%20event%20surabayajs.png "Event bulanan komunitas SurabayaJS ")

## Sesi Pertama

Sesi pertama pada event ini dibawakan oleh **[Dzulfikar Adi Putra](https://kopi.dev/2021-01-menghadiri-event-komunitas-surabayajs-di-dilo-surabaya/)** dengan materi jQuery Alternatives, bagi saya pribadi materinya yang disuguhkan oleh **Dzulfikar Adi Putra** cukup keren, Dzulfikar Adi Putra mencoba menerangkan beberapa alternative library selain jQuery, diantaranya adalah Vue.js, unpoly, alpine.js, dll. Bagi yang belum memiliki kesempatan datang di acara event komunitas SurbayaJS kali ini, teman-teman dapat membaca materinya di [https://jquery-alt.pikarlabs.com](https://jquery-alt.pikarlabs.com/demo/tabs)

![Dzulfikar adi putra di Event bulanan komunitas SurabayaJS ](./images/Dzulfikar%20membawakan%20acara.png "Dzulfikar adi putra di Event bulanan komunitas SurabayaJS ")

## Sesi Kedua

Sesujurnya pada sesi kedua saya keluar dari venue, karena ada hal urgent yang harus saya lakukan. Jadi saya tidak dapat bercerita banyak tentang sesi kedua ini, tapi teman-teman dapat menikmat tulisan **Rakha Asyrofi** di [https://medium.com/@asyrofist/machine-learning-dengan-javascript-6f3a461ac748](https://medium.com/@asyrofist/machine-learning-dengan-javascript-6f3a461ac748) lengkap banget.

![Rakha Asyrofidi Event bulanan komunitas SurabayaJS ](./images/rakha%20di%20Event%20bulanan%20komunitas%20SurabayaJS%20.png "Rakha Asyrofi Event bulanan komunitas SurabayaJS ")
